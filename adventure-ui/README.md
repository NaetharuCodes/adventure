# Adventure

Adventure is a web based RPG game inspired by both tabletop RPG games and text based adventures from the 1980s.

## Modes

Adventure supports two modes:

- Narrative Mode sees you take on a specific story.
- Open allows you to travel the land of Adventure and explore at your own pace.

## Systems

- Conversation (Dialogue options)
- Mechanical (Simple D6)
- Navigation (overland and dungeon)
- Loot / Inventory
- Magic
- Sanity
